/*INSTRUÇÕES PARA USO DA APLICAÇÃO WEB DE UM CHAT UTILIZANDO SOCKET*/
DESENVOLVIDO POR ADRIANO YUJI SATO DE VASCONCELOS COMO TRABALHO PRÁTICO
 DA DISCIPLINA DE REDES DE COMPUTEDORES

INSTRUÇÕES

PRIMEIRO PASSO:
	É necessario a instalação do framework NodeJS e do NPM que pode ser efetuada
	pelo site https://nodejs.org/en/download/ ou através dos comandos de linha:

	Para Windows é necessario instalar o package manager Chocolatey antes e depois
	inserir "cinst nodejs.install" no terminal.

	Ubuntu e Debian
	# Using Ubuntu
	curl -sL https://deb.nodesource.com/setup_13.x | sudo -E bash -
	sudo apt-get install -y nodejs

	# Using Debian, as root
	curl -sL https://deb.nodesource.com/setup_13.x | bash -
	apt-get install -y nodejs

SEGUNDO PASSO:
	Deve se abrir o diretório .\trabalho-socket no terminal e executar o comando 
	"npm run dev" e aguardar a inicialização do servidor.

TERCEIRO PASSO:
	Agora só é necessário executar o arquivo index.html para usufruir da aplicação!
	Toda vez que o arquivo for iniciado, um novo usuário ira se conectar ao servidor
